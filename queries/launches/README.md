# Consulta para obtener todas las imagenes de las launches

**Method** : `GET`

**Version** : `v3`

**URL** : `https://api.spacexdata.com/v3/launches`

**Storage** : `v3.js`

**Modulo** : `getAllImgLaunches`

**Auth required** : `False`

## Success Responses

**Code** : `200 OK`
